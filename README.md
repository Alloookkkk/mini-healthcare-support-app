# Mini Healthcare Support Web App

## Overview
This is a simple concept-level healthcare support portal for NGOs.
It collects patient support requests and provides basic automated assistance.

## Features
- Healthcare support form
- Auto-response after form submission
- FAQ chatbot concept for common questions

## AI / Automation Idea
The app includes a basic automation feature where users receive instant
confirmation after submitting the form. A chatbot concept is also included
to answer frequently asked questions, which can later be extended using AI APIs.

## Tech Stack
- HTML
- CSS
- JavaScript

## NGO Use Case
NGOs can use this app to easily collect healthcare requests, manage volunteers,
and reduce response time using automation.
